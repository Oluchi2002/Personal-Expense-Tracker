# Personal Expense  Tracker

## Overview

The Personal Expense Tracker is a desktop-based financial management application developed using Python, Tkinter, SQLite, and tkcalendar. The application enables users to record, store, and monitor their daily expenses through a simple and interactive graphical user interface.

The system automatically calculates daily, monthly, and annual spending totals while securely storing transaction records in a local SQLite database.

---

## Features

* Record new expenses
* Select transaction dates using a calendar widget
* Store expense records in SQLite
* Track multiple payment methods
* View transaction history in a tabular format
* Calculate total spending:

  * Daily
  * Monthly
  * Annual
* User-friendly graphical interface
* Real-time dashboard updates

## Installation

### Clone the Repository

```bash
git clone https://github.com/Oluchi2002/Personal-Expense-Tracker.git
```

### Navigate into the Project Folder

```bash
cd Personal-Expense-Tracker
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run the Application

```bash
python expense_tracker.py
```
